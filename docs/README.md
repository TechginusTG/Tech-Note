# Tech-Note: 풀스택 블로깅 플랫폼

Tech-Note는 개발자와 기술 애호가를 위해 설계된 모던 풀스택 블로깅 플랫폼입니다. 기술 아티클과 튜토리얼을 작성, 공유, 발견할 수 있는 깔끔하고 직관적인 인터페이스를 제공합니다. 이 프로젝트는 강력한 기술 스택을 활용하여 고성능의 확장 가능하며 풍부한 기능의 경험을 제공합니다.

## 시작하기

이 프로젝트의 개발에 참여하고 싶다면, 다음 문서들을 참고해주세요:

- **[프로젝트 개요](./docs/OVERVIEW.md):** 프로젝트의 비전, 핵심 기능, 기술 스택에 대해 더 알아보세요.
- **[로컬 개발 환경 설정](./docs/LOCAL_DEVELOPMENT.md):** 개발을 시작하기 위해 로컬 환경을 설정하는 방법을 확인하세요.
- **[백엔드 가이드](./docs/BACKEND.md):** 백엔드 아키텍처와 API 개발 방법에 대해 알아보세요.
- **[프론트엔드 가이드](./docs/FRONTEND.md):** 프론트엔드 아키텍처와 컴포넌트 개발 방법에 대해 알아보세요.
